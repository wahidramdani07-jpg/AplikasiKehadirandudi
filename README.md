# Aplikasi Kehadiran Android

Aplikasi untuk verifikasi kehadiran karyawan menggunakan kamera dan Google Maps.

## Fitur
- Ambil foto (Camera Intent)
- Input ID Karyawan
- Verifikasi data
- Tampilkan hasil
- Buka lokasi kantor (Google Maps)

## Teknologi
- Java
- Android Studio
- Intent (Implicit & Explicit)
